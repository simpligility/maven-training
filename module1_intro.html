Prerequisites

For the best learning experience this class has been designed to be both efficient and fast-paced. Here are a few prerequisites we've assumed...

You are Comfortable with the OS

In this class we ask you to perform the following tasks without much explanation:

  • Download a file to your workstation
  • Set an Environment Variable
  • Run shell and batch scripts

You Understand Java Development

This class is targeted at Java developers. We assume that you:

  • Know how to install Java.
  • Understand the structure of simple programs.
  • Have had experience compiling Java code.

Target Technologies

We've developed this class to address a set of technologies currently used for development. These technologies include...

Operating Systems

You'll have no problem running the examples in this class if you use:

  • Windows 7 or Higher
  • OSX Mountain Lion

You can also use Linux, (but you do so at your own risk.)

Java Version

This class is based on Java 7.

Can you use Java 6? One or two examples may break, use at your own risk.

IDE

The examples in this class will feature Eclipse Kepler.

But, you can use IntelliJ or NetBeans if that is what you are familiar with.

Again, use an alternative IDE at your own risk.

Goals and Objectives

At the end of this class, you should know:

  • How to download and install Maven
  • How to run a simple build
  • How to configure and locate dependencies for a project
  • How to configure the Surefire plugin to run unit tests
