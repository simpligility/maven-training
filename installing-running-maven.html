Install Java 7

Maven is designed to run on Java 6 or higher.

This class uses Java 7.

  1. Visit Oracle's JavaSE Page: http://java.oracle.com
  2. Download Java Development Kit (JDK)
  3. Install JDK
  4. Set the JAVA_HOME variable to point to the installation directory.

Downloading Maven

Downloading Maven is Easy

  1. Go to http://maven.apache.org/download.cgi
  2. Download either apache-maven-3.1.1-bin.tar.gz or apache-maven-3.1.1-bin.zip

Installing Maven

  1. Unpack the Maven distribution to a directory. (i.e. /usr/local/apache-maven-3.1.1)
  2. Set two environment variables:
    1. M2_HOME should point to /usr/local/apache-maven-3.1.1
    2. PATH add ${M2_HOME}/bin to PATH
  3. Restart shell to pick-up environment variable changes

Running Maven

Once Maven has been installed, run the following command:
