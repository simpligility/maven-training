Installing and Running Maven

First Steps

Min Time: 10 minutes, Max Time: 15 minutes

  • Instead of starting with a VERY BORING "What is Maven?" section. We start with a demo.
  • Pay attention to prerequisites - if someone doesn't get how to set an environment, they shouldn't be in this class.

Module Objectives

  • Install Java 7
  • Download Maven
  • Install Maven

  • Answer Questions...

Install Java 7

Maven is designed to run on Java 6 or higher.

This class uses Java 7.

  1. Visit Oracle's Java SE Page: http://java.oracle.com
  2. Download Java Development Kit (JDK)
  3. Install JDK
  4. Set the JAVA_HOME variable to point to the installation directory

Downloading Maven

.. is Easy

  1. Go to http://maven.apache.org/download.cgi
  2. Download zip or tar.gz file, contents are identical
    • apache-maven-3.2.3-bin.tar.gz
    • apache-maven-3.2.3-bin.zip

Installing Maven

  1. Unpack the Maven distribution in any directory
    • e.g. extraction in /opt will create /opt/apache-maven-3.2.3
  2. Set two environment variables:
    • M2_HOME should point to /opt/apache-maven-3.2.3
    • PATH add ${M2_HOME}/bin to PATH
  3. Restart shell to pick-up environment variable changes

Running Maven

Once Maven has been installed, run the following command:
