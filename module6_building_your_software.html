Ways to Configure a Goal

  • From the command-line
  • In a Goal Execution
  • In the plugin configuration
  • These different levels are used for different reasons.
  • Command-line customizations are one-off, throw away commands.
  • Configuration at the execution level are for targeted changes.
  • Configuring at the Plugin is for wider customization.
